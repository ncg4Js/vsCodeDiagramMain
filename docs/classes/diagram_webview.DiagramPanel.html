    Class DiagramPanel

    Singleton webview panel that displays the Genero BDL call-graph diagram.

    Only one panel is kept alive at a time. Calling getOrCreate returns the existing instance (and reveals it) if one already exists, or creates a new one otherwise.

    The panel communicates with its HTML content via postMessage / onDidReceiveMessage:

    • Outgoing (updateGraph): pushes a fresh ASCII diagram and statistics string.
    • Incoming (navigate, refresh, export, exportHtml, cancel): user actions from the toolbar buttons and Mermaid node clicks.
